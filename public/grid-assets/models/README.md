# Grid Ops Model Placeholders

This directory is reserved for low-poly GLB assets used by the 2.5D Grid Ops scene.

Recommended initial files:
- `control-center.glb`
- `transformer.glb`
- `solar-array.glb`
- `battery-storage.glb`
- `frequency-controller.glb`

The scene currently renders icon-first procedural nodes and safely falls back when model files are unavailable.
